# Icon Placeholder Instructions

To complete the PWA setup, you need to add icon files:

1. Create `icon-192.png` (192x192 pixels)
2. Create `icon-512.png` (512x512 pixels)

Place both files in the `public/` directory.

You can:
- Use an online icon generator
- Create icons with your app logo
- Use a design tool like Figma/Canva

The icons should represent "Apni Dukan" - a grocery/shopping theme works well.

For now, the app will work without icons, but they're needed for a complete PWA experience.
